Définition du HTML

HTML (HyperText Markup Language) est ce qu'on appelle un langage de balisage.
Créé en 1991 par Tim Berners-Lee, HTML est le langage de balisage standard utilisé pour créer des pages web.
Il permet de structurer et d'organiser le contenu d'une page web en utilisant des balises.

Principe du balisage

HTML utilise des balises pour définir les différents éléments d'une page web.
Une balise HTML est entourée de crochets (< et >) et elle est généralement utilisée par paires : une balise ouvrante et une balise fermante.

Les balises sont utilisées pour créer des éléments tels que des titres, des paragraphes, des liens, des images, et bien plus encore.

Balises couramment utilisées

  • <p> : paragraphe ;
  • <h1> à <h6> : titres (h1 pour le titre le plus important, h6 pour le moins important) ;
  • <img src="url" alt="description"> : image ;
  • <a href="url"> : lien ;
  • <ul> et <ol> : listes (ul pour liste non ordonnée, ol pour liste ordonnée) ;
  • <li> : élément de liste.

Structure d'une page HTML

Une page HTML est composée de deux sections principales : l'en-tête (head) et le corps (body).

  • <head> : cette section contient des informations sur la page, telles que le titre, les liens vers les feuilles de style, et les métadonnées.
  • <body> : il s'agit du contenu visible de la page, comme les textes, les images, et les autres éléments. Le body comprend généralement les sections suivantes :
    • <header> : contient le titre et le menu de navigation de la page ;
    • <main> : contient le contenu principal de la page, organisé en sections ;
    • <footer> : contient les informations de pied de page, comme les droits d'auteur et les liens vers les mentions légales.

Pas à pas : crée ta première page HTML

Suis ces étapes pour créer une page HTML simple :

  1. ouvre un éditeur de texte comme VSCode
  2. crée un nouveau fichier et nomme-le index.html
  3. ajoute le code suivant :
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ma Première Page HTML</title>
</head>
<body>
    <header>
        <h1>Bienvenue sur ma première page HTML</h1>
    </header>
    <main>
        <section>
            <p>Ceci est un paragraphe.</p>
        </section>
    </main>
    <footer>
        <p>Ceci est le pied de page.</p>
    </footer>
</body>
</html>
  1. enregistrez le fichier
  2. ouvre le fichier dans un navigateur web pour voir le résultat

Astuces

Tu peux "inspecter" les éléments d'une page web et ainsi consulter son code avec l'outil "Inspecter" intégré dans la plupart des navigateurs. Voici comment faire :

  1. ouvre une page web dans un navigateur
  2. fais un clic droit
  3. sélectionne "Inspecter"
  4. une nouvelle fenêtre ou un panneau s'ouvrira, affichant le code HTML
