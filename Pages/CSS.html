Définition du CSS

CSS (Cascading Style Sheets) est un langage de style utilisé pour décrire la présentation d'un document HTML.
Il permet de contrôler l'apparence des éléments sur une page web.

Fonctionnement du CSS

Le CSS utilise des boîtes pour modéliser les éléments. Chaque élément HTML est considéré comme une boîte.
Les boîtes peuvent être mises en page en utilisant des modèles comme Flexbox et Grid.

Flexbox

Flexbox est un modèle de mise en page unidimensionnelle qui permet de répartir l'espace entre les éléments d'une boîte.

Grid

Grid est un modèle de mise en page bidimensionnelle qui permet de créer des grilles de colonnes et de lignes pour organiser les éléments.

Propriétés CSS

Propriété Description
color Définit la couleur du texte
background-color Définit la couleur de fond
font-size Définit la taille de la police
margin Définit la marge extérieure d'un élément
padding Définit l'espace intérieur d'un élément

Pas à pas : crée ton premier fichier CSS

Suis ces étapes pour créer une page HTML simple :


  1. ouvre dans ton éditeur de texte préféré le dossier qui contient ton fichier index.html
  2. crée un nouveau fichier et nomme-le style.css
  3. lie ton fichier CSS à ton fichier HTML en en utilisant la balise <link> dans le <head> de ton fichier HTML
  4. ajoute le code suivant dans ton fichier CSS pour donner du style à ton projet :

html {
    font-family: Arial, sans-serif;
}

body {
    margin: 0;
    padding: 0;
    background-color: #f4f4f4;
}

header {
    background: #333;
    color: #fff;
    padding: 10px 0;
    text-align: center;
}

header h1 {
    margin: 0;
}

main {
    padding: 20px;
}

section {
    background: #fff;
    padding: 20px;
    margin: 10px 0;
    border-radius: 5px;
    box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
}

footer {
    background: #333;
    color: #fff;
    text-align: center;
    padding: 10px 0;
    position: fixed;
    width: 100%;
    bottom: 0;
}

p {
    margin: 0;
    padding: 10px 0;
}
  1. enregistrez le fichier
  2. ouvre le fichier dans un navigateur web pour voir le résultat
  3. tadaaa !
