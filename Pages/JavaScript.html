Définition du JavaScript

JavaScript est un langage de programmation utilisé pour créer des pages web interactives.
Il permet de modifier le contenu d'une page web en fonction des actions de l'utilisateur.

Fonctionnement du JavaScript

Le JavaScript fonctionne en manipulant le DOM (Document Object Model). Le DOM est une représentation de la structure de la page HTML sous forme d'arborescence.

DOM

Le DOM permet d'accéder et de manipuler les éléments HTML à l'aide de JavaScript.

Événements

Le JavaScript réagit aux actions de l'utilisateur en utilisant des événements, comme les clics de souris ou les saisies clavier.

Propriétés JavaScript

Méthode Description
document.getElementById() Récupère un élément HTML par son ID
addEventListener() Attache un événement à un élément
innerHTML Modifie le contenu HTML d'un élément
style Modifie les styles CSS d'un élément

Pas à pas : crée ton premier fichier JavaScript

Suis ces étapes pour créer une page JavaScript simple :

  1. ouvre dans ton éditeur de texte préféré le dossier qui contient ton fichier index.html
  2. crée un nouveau fichier et nomme-le script.js
  3. lie ton fichier JavaScript à ton fichier HTML en utilisant la balise <script> avant la fermeture de la balise <body> :

<script src="script.js"></script>
  1. ajoute le code JavaScript suivant dans ton fichier script.js :

function afficherMessageBienvenue() {
    alert("Bienvenue sur easyDev !");
}

window.onload = function() {
    afficherMessageBienvenue();
};
  1. enregistre les fichiers
  2. ouvre le fichier HTML dans un navigateur web pour voir le résultat
  3. tadaaa !
